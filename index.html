▶? How to use The Ledger

Today

Log time blocks with a label, category, start time, and duration. They appear on the vertical day timeline and the totals wheel. Use the date arrows or picker to jump to other days. Log sleep here too — just use the "Sleep" category and it'll roll into your dashboard stats automatically.

Goals

Set a target and track progress toward it. Progress bars and the dashboard's overall percentage update as you update each goal.

Weight

Add weigh-ins over time to see your trend on a simple line chart.

Dashboard

A weekly and monthly overview of what happened: time by category, sleep averages, goal progress, and a weight trend — all objective, no advice.

Insights

Your coach — turns this week's logged data into plain-language cards (biggest win, an opportunity, a hidden pattern, and more). It remembers what it told you last time and follows up ("Since Last Time" cards, plus a running Coaching Thread below your cards) instead of starting fresh every refresh. Works offline with a built-in local coach; add a Gemini or Mistral API key under "Providers" to have an AI write the phrasing instead.

Coach Profile

Click "Coach Profile" in the top bar to tell Ask Ledger who you are — identity, goals, routine, and things to avoid — so it stops treating every question like a cold start. It also keeps a running Coach Memory of stable patterns it notices over time (e.g. "mornings are your best focus window"), which you can review or delete there. Only the most relevant memories are sent with each question, not the whole list. Use "← Back" to return to whatever tab you were on.

Saving

Changes save automatically as you go. Use Save in the top bar to force-save immediately, or Reset Day to clear all time blocks for the day currently shown.

Tip: everything is stored locally per day/entry, so you can revisit past dates any time from the Today tab.
